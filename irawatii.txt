Mengenal tentang Git :
Git adalah salah satu sistem pengontrol versi (Version Control System) pada proyek perangkat lunak yang diciptakan oleh Linus Torvalds.Git merupakan salah satu tool yang sering digunakan dalam proyek pengembangan software.

Git dikenal juga dengan distributed revision control(VCS terdistribusi), artinya penyimpanan database Git tidak hanya berada dalam satu tempat saja.
Dalam Git terdapat merge untuk menyebut aktifitas penggabungan kode.

Apa yang dilakukan oleh Git ?
Git sebenarnya akan memantau semua perubahan yang terjadi pada file proyek.Lalu menyimpannya kedalam database.

Kenapa Git penting bagi programmer ?
Untuk menguasai jutsu kolaborasi, maka kita harus bisa menggunakan Git.Jadi selain untuk mengontrol versi, git juga digunakan untuk kolaborasi.Saat ini Git menjadi salah satu tool terpopuler yang digunakan pada pengembangan software open souce maupun closed source.

Cara membatalkan revisi menggunakan perintah git checkout, git reset, dan git revert.

1. Git Checkout = Perintah git checkout seperti mesin waktu, kita bisa kembalikan kondisi file proyek seperti waktu yang dituju.
2. Git Reset = Perintah git reset sering disebut sebagai perintah berbahaya yang dapat menghancurkan catatan sejarah perubahan.
3. Git Revert = Perintah ini lebih aman daripada git reset, karena tidak akan menghapus catatan sejarah revisi.
